WeatherOS

WeatherOS is a modern and responsive weather dashboard that provides real-time weather information with an elegant, colorful user interface. It allows users to search weather conditions for cities worldwide, view hourly and 7-day forecasts, monitor air quality, UV index, wind details, sunrise and sunset times, and much more.

Weather data is powered by the OpenWeather API and presented through an interactive dashboard with smooth animations and responsive design.

Features

- Real-time weather information
- Search weather for any city worldwide
- Automatic location detection
- 24-hour weather forecast
- 7-day weather forecast
- Interactive temperature trend chart
- Air Quality Index (AQI)
- UV Index with safety recommendations
- Wind speed and compass direction
- Humidity, pressure and visibility
- Sunrise and sunset information
- Cloud cover and dew point
- Responsive design for desktop, tablet and mobile
- Smooth animations and colorful user interface

Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Tailwind CSS
- Font Awesome
- Google Fonts
- OpenWeather API
